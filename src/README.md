# ShadowFox Banking App

Bank Account Management System with JUnit Testing
built as part of ShadowFox Java Internship - Part 2 Task 1.

## Features
- Create bank accounts
- Deposit money
- Withdraw money
- Check balance
- Mini statement with timestamps
- Transaction history
- 10 JUnit tests passing ✅

## How to Run

### Banking App
1. Open in IntelliJ
2. Run `BankingApp.java`

### Run JUnit Tests
1. Open in IntelliJ
2. Right-click `BankAccountTest.java`
3. Click Run Tests
4. All 10 tests should pass ✅

## Technologies Used
- Java 25
- JUnit 5
- BigDecimal
- Maven
